void MegaNode.AddNode ( Node  n)

Adds a new node to the mega node Mass is increased, and the node is added to containedNodes. Relative position is also saved, but it's likely to be wonky, not that it matters too much. The new node is also disabled (simulating = false), and gets an index pointer to this mega node.
